## 安装

```shell
tnpm i @alipay/oasis-run -g
```

## 使用


### 1. link

在项目根目录下执行：

```shell
oar link
```

**第一次使用输入 Oasis3D 项目根目录。**


### 2. create

创建 Oasis 二方库：

```shell
oar create
```

Oasis Engine 发布，在 Oasis Engine 目录下执行：

```shell
oar release
```

## 安装

```shell
tnpm i @alipay/oasis-run -g
```

## 使用


### 1. link

在项目根目录下执行：

```shell
oar link
```

**第一次使用输入 Oasis Engine 项目根目录。**

如果引擎目录变化，使用：

```shell
oar link -c
```

### 2. create

创建 Oasis 二方库：

```shell
oar create
```

![oar](https://gw.alipayobjects.com/zos/OasisHub/b41756d2-9e06-45b1-a533-4a91decde9c0/oar.gif)


### 3. release

Oasis Engine 发布，在 Oasis Engine 目录下执行：

```shell
oar release
```

## Usage

首先安装 [@alipay/maf](https://web.npm.alibaba-inc.com/package/@alipay/maf)：

```bash
tnpm install @alipay/maf -g
```

### 从 npm 拉取

```bash
maf create @alipay/maf-generator-oasis-ext-template
```

### 从 repo 拉取

```bash
maf create maf-generators/oasis-ext-template
```
